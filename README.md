# Team Attendance Tracker

A web-based attendance tracker for VQC and Enrichment teams (20 members).

## Live Dashboard
https://roshanraj-sys.github.io/attendance-tracker

## Features
- Member check-in with WFH/WFO mode selection
- Leave and overtime request management
- Admin dashboard with approvals
- Monthly attendance report with CSV export
- Dark mode support
- Firebase Firestore backend (real-time, free)
